About the PHP port for Android

This is app has built-in PHP server(the one that is available in PHP since 5.4 version) compiled for Android ARM architecture, and can be started automatically on Android boot.

PHP(version 5.4.15) has theese modules: bcmath, bz2, dom, ereg, fileinfo, ftp, gd, hash, iconv, intl, libxml, mbstring, mysql, mysqli, mysqlnd, openssl, pcre, pdo, pdo_mysql, pdo_sqlite, simplexml, sqlite, sqlite3, xml, xmlreader, xmlwriter, xsl, zip, zlib; phar, soap, sockets, reflection, session, tokenizer, spl, filter, ctype, date, json; also has proper support for DNS.

Recommended for usage: in development or for very very small websites with very small amount of visitors, couse this server supports only one request at a time.

Server document root by default(it can be changed) is sdcard/www so put files there.

Edit php.ini and resolv.conf as needed, they are loaded from document root directory.

If openssl is used - don`t forget to create and use custom openssl.cnf file.

Sorry but standard PHP locales wont work(setlocale and such).
